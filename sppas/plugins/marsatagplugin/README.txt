brief:      SPPAS plugin for MarsaTag.

MarsaTag is a Part-Of-Speech tagging for French developed at Laboratoire
Parole et Langage, by S. Rauzy and G. De Moncheuil.
This plugin allows to run MarsaTag on time-aligned tokens generated
by the automatic annotation 'Alignment' of SPPAS.

MarsaTag is available for download at:
https://www.ortolang.fr/market/tools/sldr000841
